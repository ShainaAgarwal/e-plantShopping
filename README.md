# e-plantShopping 

e-plantShopping is a React-based web application for an online plant store called **Paradise Nursery**.
The website allows users to browse different categories of plants and add them to a shopping cart.

## Features

* Browse plants by category
* View plant details including image, description, and cost
* Add plants to the shopping cart
* Update quantity of items in the cart
* Remove items from the cart
* View total cart amount
* Continue shopping or proceed to checkout

## Technologies Used

* React.js
* Redux Toolkit
* JavaScript
* HTML & CSS

## Project Structure

* **ProductList.jsx** – Displays all plant products
* **CartItem.jsx** – Manages items in the shopping cart
* **CartSlice.js** – Redux slice for cart state management
* **store.js** – Redux store configuration

## How It Works

Users can browse plants and click **Add to Cart**.
Redux manages the cart state globally, allowing users to update quantities, remove items, and see the total price dynamically.
